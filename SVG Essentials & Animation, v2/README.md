## Course Review

The course provided an in-depth understanding of SVG, the versatile graphics format for the web. From debunking myths
about SVG support to exploring its powerful capabilities in improving website performance, the content was
well-structured and easy to follow. The instructor, Sarah, did an excellent job introducing each topic with clarity and
enthusiasm.

I recently completed the course on building and optimizing SVG, and I must say it was an enlightening journey into the
world of scalable web graphics. Although the course was published on January 14, 2019, it delivered a comprehensive and
hands-on learning experience that exceeded my expectations, and I think the course is also relevant today.

In conclusion, this course offers a well-rounded education on SVG, catering to beginners and experienced developers
alike. The instructor's passion for the subject matter shines through, making the learning experience engaging and
enjoyable. Whether you're aiming to enhance your web development skills or seeking to create visually stunning websites,
this course is a must-have in your learning arsenal.

## Course contents

| #  | Topic                                      | Overview                                                                                                                                         |
|----|--------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| 1  | Introduction & Setup                       | Sarah gives an introduction, sets up the course, and reviews Codepen setup and Greensock plugins.                                                |
| 2  | What is SVG?                               | Sarah provides an overview of what SVG is and its capabilities.                                                                                  |
| 3  | SVG Support & Performance                  | Debunks the belief that SVG is not widely supported and discusses its impact on website performance.                                             |
| 4  | The Antidote to Positioning in CSS         | Sarah argues that SVG eases positioning in CSS and showcases its use in loaders.                                                                 |
| 5  | Platonic Shapes                            | Breakdown of rectangle, circle, polygon, and line platonic SVGs.                                                                                 |
| 6  | viewBox & Responsive                       | Explanation of what the viewBox is and how it reacts to changes in width.                                                                        |
| 7  | preserveAspectRatio                        | Introduction to preserveAspectRatio's meet, slice, and none parameters.                                                                          |
| 8  | Paths, Groups, & Polylines                 | Introduction to open and closed paths, groups, and polylines.                                                                                    |
| 9  | Animated Bézier Curves & Template Literals | Resource on understanding animated Bézier curves and using template literals for animated lines.                                                 |
| 10 | Accessibility                              | Details on making SVG readable to screen readers and additional resources.                                                                       |
| 11 | Optimizing & Building                      | Methods to reduce SVG file size and optimize with many data points.                                                                              |
| 12 | Constructing an SVG                        | Effective grouping of objects for animation in Adobe Illustrator.                                                                                |
| 13 | Starting an SVG                            | Walkthrough of the process of starting an SVG.                                                                                                   |
| 14 | Prototyping                                | Advantages of low-resolution prototyping.                                                                                                        |
| 15 | Planning an Animation Exercise             | Students instructed to create 3 thumbnails as a storyboard from an SVG in the repo.                                                              |
| 16 | Planning an Animation Review               | Review and discussion of student creations.                                                                                                      |
| 17 | Animation Support Breakdown                | Quick refresher on CSS animation keyframes and warnings.                                                                                         |
| 18 | Differences in Functionality               | Highlighting key differences in property names and functionality between SVG and CSS.                                                            |
| 19 | Why SVG?                                   | Motivation for using SVG in web drawing.                                                                                                         |
| 20 | Transform-Origin                           | Demonstration that SVG rotates around a different axis by default than assumed.                                                                  |
| 21 | Animate an SVG Exercise                    | Students instructed to animate the storyboard created in the previous exercise.                                                                  |
| 22 | SVG Tools and Ideas Q&A                    | Questions asked about open source SVG animation software and generating ideas for SVGs.                                                          |
| 23 | Sprites                                    | Brief discussion on SVG Sprites and their usage.                                                                                                 |
| 24 | Atmospheric & Elemental Motion             | Explanation of how elemental motion allows animating seemingly complicated images.                                                               |
| 25 | Tools Overview                             | Discussion on pros and cons of DOM, Canvas, CSS/SCSS, GSAP, and React Spring/React-Motion.                                                       |
| 26 | Performance & GSAP                         | Discussion on hardware acceleration versus control and how GSAP improves workflow.                                                               |
| 27 | TweenMax Syntax                            | Introduction to basic GSAP syntax.                                                                                                               |
| 28 | Stagger                                    | Comparison of stagger implementation in CSS, SASS, and GSAP.                                                                                     |
| 29 | GSAP Monster Demo                          | CodePen demonstration of utilizing the TweenMax library.                                                                                         |
| 30 | Cycle Stagger                              | Walkthrough of code to produce cycle stagger.                                                                                                    |
| 31 | Setting CSS Properties                     | Discussion on setting CSS properties in GSAP and its advantages.                                                                                 |
| 32 | Comments on D3                             | Brief discussion on using D3 with GreenSock.                                                                                                     |
| 33 | Timeline                                   | Discussion on what timelines can do, the position parameter, nesting timelines, prevent momentary display, and percentage-based transforms.      |
| 34 | GSAP Demo                                  | Live coding of the animation of several boxes using GreenSock.                                                                                   |
| 35 | GSAP Exercise                              | Students instructed to animate an SVG with GreenSock.                                                                                            |
| 36 | UI vs Standalone Animation                 | Discussion on the purpose of UI Animation versus standalone animation.                                                                           |
| 37 | Social Engineering with Animation          | Explanation of using animation to improve the user experience with anticipatory cues and color accents.                                          |
| 38 | Context Switching                          | Examples where animation helps to switch the context of what the user is viewing.                                                                |
| 39 | Improving an Existing UI Demo              | Demonstration of how a travel website's UI could be improved with animation.                                                                     |
| 40 | Interaction, JS Detection, & Scaling       | Importance of showing differences between states, using JavaScript to detect when animation should start, and applying animations across a site. |

| 42 | Interaction Demo                           | Demonstration of using interaction and timeline functions to create a machine putting a teddy bear together.                                     |
| 43 | Draggable                                  | Introduction to a plugin allowing actions such as testing if objects are touching, momentum effects, and axis locking.                           |
| 44 | Interactive Exercise                       | Students instructed to make the last exercise interactive.                                                                                       |
| 45 | GSAP Extras                                | Introduction to GSAP plugins: DrawSVG, Motion Along a Path, Curviness & Rotation, Animating Text, Relative Color Values.                         |
| 46 | DrawSVG                                    | Introduction to a plugin for progressively revealing or hiding the stroke of an SVG.                                                             |
| 47 | Motion Along a Path                        | Introduction to a plugin allowing more natural animations along a path.                                                                          |
| 48 | Curviness & Rotation                       | Introduction to parameters for curviness and rotation in GSAP.                                                                                   |
| 49 | Animating Text                             | Discussion on using SplitText and making gradients more performant.                                                                              |
| 50 | Relative Color Values                      | Demonstration of using HSL values to gradually change a color for scene transitions.                                                             |

| 55 | Clipping & Masking                         | Introduction to clipping and masking to make objects partially or wholly disappear in animations.                                                |
| 56 | viewBox                                    | Review of viewBox and how to interact with its parameters.                                                                                       |
| 57 | svgOrigin                                  | Introduction to svgOrigin as a way to move an object about an origin.                                                                            |
